## 一、核心指标与阈值
构建可重复、高胜率交易系统的第一步，是确立一套客观、量化的核心观测指标及其关键阈值。本系统以通达信涨跌家数指数（880005）为情绪周期的“锚点”与“体温计”，结合辅助指标，形成了一套精细的阈值判断标准。

### 核心观测指标体系
本系统依赖三大类指标进行综合判断：

- **核心情绪指标**：通达信涨跌家数指数（代码：880005）。它代表当日市场中上涨股票（红盘）的家数，是衡量全市场普涨/普跌情绪最直观、最客观的指标。

- **关键均线**
  - 5日移动平均线（MA5）：代表中期/整体大盘情绪的平均水平和趋势惯性，是仓位管理与周期定位的最高优先级信号。
  - 3日移动平均线（MA3）：代表短期/短线投机情绪的波动，常与5日线组合使用，判断情绪共振状态。

- **辅助判断指标**
  - 同花顺全A股CCI指标（参数设为5）：用于判断市场轮动节奏与容量票的波段交易时机，反映指数波动对趋势赚钱效应的影响。
  - 昨日炸板股指数：用于评估市场修复力度和短线容错率，是控制仓位和出手时机的重要参考。
  - 昨日涨停表现指数：作为板块强度与赚钱效应延续性的辅助观察。

---

### 关键阈值与精确定义
所有阈值均基于历史数据回测与市场生态（如上市公司总数）动态归纳得出，是触发交易动作的客观依据。

#### 1. 红盘家数（880005）的绝对阈值
这是判断市场情绪极端状态的直接标尺。

| 情绪状态 | 核心定义 | 数值范围（动态调整） | 市场特征与操作基调 |
| :--- | :--- | :--- | :--- |
| **冰点 / 极值** | 市场情绪极度低迷、恐慌的底部区域 | 早期标准（约2018-2019年，3000+家上市公司）：红盘家数 ≤ 500家<br>当前标准（5000+家上市公司）：红盘家数 ≤ 1000家（常用，约为总数的20%以内）<br>更灵敏参考：红盘家数 ≤ 1500家 | 市场特征：情绪恐慌，亏钱效应弥漫，但物极必反，是情绪向上修复的起点。<br>操作基调：进攻。寻找买点，开仓/加仓，博弈次日反弹。优先关注抗跌核心、高度板弱转强、最早反弹首板。 |
| **高潮 / 顶值** | 市场情绪极度亢奋、一致的顶部区域 | 早期标准：红盘家数 ≥ 3000家<br>当前标准：红盘家数 ≥ 4000家（常用，约为总数的80%以上） | 市场特征：市场普涨，情绪过于一致，后续情绪大概率向下回归。<br>操作基调：防守。减仓、降仓位，放弃高位接力，仅关注低位补涨或新题材首板。 |
